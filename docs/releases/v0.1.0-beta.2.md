# Crypto Lens 0.1.0 Beta 2

此版本更新了 Crypto Lens 的应用图标，采用浅色、轻扁平的 macOS 风格：冰川蓝镜片、翡翠绿趋势线与珍珠白背景。功能和数据兼容性与 Beta 1 保持一致。

支持 Apple Silicon 和 Intel Mac，最低系统版本为 macOS 14。

## 安装

1. 下载 `CryptoLens-0.1.0-beta.2.dmg`。
2. 打开 DMG，将 Crypto Lens 拖入 Applications。
3. 首次尝试启动后，打开“系统设置 → 隐私与安全性”。
4. 找到 Crypto Lens 的拦截提示，点击“仍要打开”，然后再次确认“打开”。

此版本使用 ad-hoc 签名，**没有 Apple Developer ID 签名，也没有经过 Apple 公证**。请勿关闭 Gatekeeper；使用 macOS 提供的“仍要打开”功能即可。

## 校验文件

下载同名 `.sha256` 文件后，可在终端执行：

```bash
shasum -a 256 -c CryptoLens-0.1.0-beta.2.dmg.sha256
```

```text
cea386a7aac60fbca494959f13fb5687df985d54c9e0d7370f78e731474e083b
```

## 主要能力

- 菜单栏快速查看加密资产与股票代币价格。
- CoinMarketCap Keyless 默认可用，可选个人 API Key。
- 本地关注列表、行情缓存、排序和批量撤销删除。
- 打开面板时刷新一次，不进行后台轮询。
- 支持断网查看最后一次成功行情。

股票代币的搜索和价格来自 CoinMarketCap；应用内置目录仅用于类型识别，不代表官方认证或投资背书。
