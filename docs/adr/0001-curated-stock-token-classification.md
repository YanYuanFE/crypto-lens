# Classify Stock Tokens with Bundled Curated Data

Crypto Lens uses the validated, bundled `CuratedStockTokens.json` as the positive Stock Token classification source. The catalog keeps its issuer and CoinGecko verification provenance, while runtime assets now come from CoinMarketCap. Legacy assets match by their persisted CoinGecko slug; CMC assets match by verified platform/contract when available or by an exact curated symbol. Classification is frozen when an Asset enters the Watchlist; if the curated resource is missing or invalid at runtime, the app logs a fault, reports Classification Unavailable once, and conservatively classifies unconfirmed Assets as Crypto Assets without blocking prices, search, or Watchlist operations.
