# Classify Stock Tokens with Bundled Curated Data

Crypto Lens uses the validated, bundled `CuratedStockTokens.json` as the only positive Stock Token classification source in v1 because CoinGecko search results do not carry sufficient category data, per-result detail requests create an N+1 path, and symbol heuristics create false positives. The catalog aims for verified, near-complete release-time coverage of CoinGecko-listed Backed xStocks and Ondo Global Markets assets rather than an arbitrary sample count. Classification is frozen when an Asset enters the Watchlist; if the curated resource is missing or invalid at runtime, the app logs a fault, reports Classification Unavailable once, and conservatively classifies unconfirmed Assets as Crypto Assets without blocking prices, search, or Watchlist operations.
